B.E. Course Structure

Comprehensive 4-year engineering program

First Year (FE)

  • Engineering Mathematics: Calculus, Linear Algebra, Differential Equations
  • Physics: Mechanics, Waves, Thermodynamics
  • Chemistry: Organic, Inorganic Chemistry
  • Programming: Introduction to C++
  • English: Technical Communication

Second Year (SE)

  • Data Structures: Arrays, Linked Lists, Trees, Graphs
  • Database Management: SQL, Relational Models
  • Web Development: HTML, CSS, JavaScript
  • Operating Systems: Process Management, Memory Management
  • Computer Architecture: CPU Design, Memory Hierarchy

Third Year (TE)

  • Advanced Algorithms: Sorting, Searching, Dynamic Programming
  • Software Engineering: SDLC, Design Patterns
  • Artificial Intelligence: Machine Learning Basics
  • Networking: TCP/IP, Routing, Security
  • Project Management: Tools and Methodologies

Fourth Year (BE)

  • Cloud Computing: AWS, Azure Basics
  • DevOps: Docker, Kubernetes
  • Cybersecurity: Encryption, Authentication
  • Capstone Project: Real-world application development
  • Internship: Industry experience
