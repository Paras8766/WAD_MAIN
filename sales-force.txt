1. Login to Salesforce

2. Click ⚙️ (top right) → click **Setup**

3. In Setup, go to **Object Manager**

4. Click **Create → Custom Object**

5. Create first object:

   * Label: Student
   * Click Save

6. In Student object → go to **Fields & Relationships → New**

7. Create fields in Student:

   * Roll Number → Number → Next → Save
   * Email → Email → Next → Save
   * Phone → Phone → Next → Save

8. Go back to Object Manager

9. Click **Create → Custom Object**

10. Create second object:

    * Label: Course
    * Save

11. Add fields in Course:

    * Course Name → Text → Save
    * Duration → Number → Save
    * Faculty Name → Text → Save

12. Again go to Object Manager

13. Click **Create → Custom Object**

14. Create third object:

    * Label: Enrollment
    * Save

15. Add fields in Enrollment:

    * Enrollment Date → Date → Save
    * Status → Picklist (Present, Absent) → Save

16. In Enrollment → go to **Fields & Relationships → New**

17. Create relationship:

    * Select Lookup Relationship
    * Choose Student → Next → Save

18. Again click **New**

19. Create second relationship:

    * Lookup Relationship
    * Choose Course → Next → Save

20. Go to Setup search → type **Tabs**

21. Open Tabs → under Custom Object Tabs click **New**

22. Create tabs:

    * Student tab
    * Course tab
    * Enrollment tab

23. Go to **App Manager**

24. Click **New Lightning App**

25. Enter:

    * App Name: Student Management System

26. Click Next until Navigation Items screen

27. Add:

    * Student
    * Course
    * Enrollment

28. Select profile:

    * System Administrator

29. Click Save

30. Click App Launcher (🔲)

31. Search and open:

    * Student Management System

32. Click Student tab → New → add student → Save

33. Click Course tab → New → add course → Save

34. Click Enrollment tab → New

35. Select Student and Course → add date and status → Save



42. Application complete and tested
