B.E. Course Structure

First Year (FE)

  • Engineering Mathematics-I

    Calculus, Sequences, and Series for engineering applications.

  • Engineering Physics

    Fundamentals of optics, semiconductors, and quantum mechanics.

  • Programming and Problem Solving

    Introduction to Python programming and logic building.

Second Year (SE)

  • Discrete Mathematics

    Logic, set theory, and graph theory foundations.

  • Data Structures and Algorithms

    Detailed study of linear and non-linear data structures.

  • Computer Graphics

    Algorithms for 2D and 3D object representation.

Third Year (TE)

  • Database Management Systems

    SQL, NoSQL, and database design principles.

  • Computer Networks

    Layered architecture and communication protocols.

  • Web Application Development

    Full-stack development using modern frameworks.

Fourth Year (BE)

  • Design and Analysis of Algorithms

    Complexity classes and advanced algorithm strategies.

  • Artificial Intelligence

    Heuristic search, neural networks, and expert systems.

  • High Performance Computing

    Parallel processing and distributed systems architecture.
